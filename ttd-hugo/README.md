# ttd-hugo

[Alpine](https://alpinelinux.org/) based container for [Hugo](http://gohugo.io/).

## Dependencies

- [Docker](https://docker.com "Homepage of docker")

## Installation

Pull the image:

```
docker pull testthedocs/ttd-hugo
```
